## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Abhrajit-2005/BasketBall_Matches.git
    cd Basketball_Matches

2. Setup the Server:
   ```bash
    cd server
    npm install
    npm start

4. Setup the Client:
   ```bash
    cd client
    npm install
    npm start

## Access the App

API_URL: https://v1.basketball.api-sports.io/

Client: http://localhost:3000

Server: http://localhost:5000
